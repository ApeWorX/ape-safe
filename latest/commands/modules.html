Modules

ape safe modules

Commands for handling safe modules

ape safe modules [OPTIONS] COMMAND [ARGS]...

disable

Disable MODULE for SAFE

WARNING: This is a potentially destructive action and may impact operations of your safe.

ape safe modules disable [OPTIONS] SAFE MODULE

Options

-v, --verbosity <LVL>

One of ERROR, WARNING, SUCCESS, INFO, or DEBUG

--account <account>
--propose

Arguments

SAFE

Required argument

MODULE

Required argument

enable

Enable MODULE for SAFE

WARNING: This is a potentially destructive action and may make your safe vulnerable.

ape safe modules enable [OPTIONS] SAFE MODULE

Options

-v, --verbosity <LVL>

One of ERROR, WARNING, SUCCESS, INFO, or DEBUG

--account <account>
--propose

Required argument
